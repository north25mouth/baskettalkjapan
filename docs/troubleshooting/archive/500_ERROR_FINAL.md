# 500エラー - 最終確認事項

## 現在の状況

- ビルドは成功している
- 開発サーバーは起動している
- しかし、`http://localhost:3000/` にアクセスすると500エラーが発生

## 確認手順

### 1. 開発サーバーのターミナルでエラーログを確認

開発サーバーを起動しているターミナルで、以下のようなエラーメッセージを探してください：

```
Error: ...
TypeError: ...
ReferenceError: ...
```

### 2. ブラウザのコンソールでエラーを確認

ブラウザでF12キーを押して、コンソールタブでエラーメッセージを確認してください。

### 3. ネットワークタブでエラーを確認

ブラウザでF12キーを押して、ネットワークタブで`/`のリクエストを確認し、レスポンスの詳細を見てください。

## 一時的な修正

現在、以下のファイルを最小限に変更しています：

- `app/page.tsx` - 最小限のコンテンツ
- `app/layout.tsx` - フォントとCSSを無効化
- `app/globals.css` - フォント変数を無効化

## 次のステップ

1. **開発サーバーのターミナルでエラーメッセージを確認**
   - エラーメッセージの全文をコピーしてください

2. **エラーメッセージを共有**
   - エラーメッセージを共有していただければ、原因を特定して修正できます

3. **開発サーバーを再起動**
   ```bash
   # 開発サーバーを停止（Ctrl+C）
   rm -rf .next
   npm run dev
   ```

## よくある原因

1. **Firebase初期化エラー**
   - 環境変数が正しく設定されていない
   - Firebase設定に問題がある

2. **TypeScriptエラー**
   - 型の不一致
   - 未定義の変数

3. **Next.jsの設定エラー**
   - `next.config.ts`に問題がある
   - ビルドキャッシュの問題

4. **依存関係の問題**
   - `node_modules`に問題がある
   - パッケージのバージョンの不一致
